# Requirements Archive: v1.0 — Contrastive Learning Tutorial Repo

**Archived:** 2026-05-03
**Milestone:** v1.0 MVP
**Core Value:** Any contrastive learning method can be implemented by conforming to a shared interface and immediately work with the same dataset pipeline, timm backbone, and Lightning training loop.

- **Total v1 requirements:** 40

All 40 requirements were implemented and unit-tested. The 6 partial requirements have integration bugs in `train.py` (3 bugs) and `eval/` scripts (1 shared bug pattern) — all deferred to v1.1 as known tech debt.
