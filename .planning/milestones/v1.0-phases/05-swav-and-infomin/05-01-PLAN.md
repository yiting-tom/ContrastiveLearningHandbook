  <action>
Per D-01, D-02, D-03: Add to core/data.py after the IndexedDataset class:

1. `MultiCropDataset(torch.utils.data.Dataset)`:
   - `__init__(self, dataset, n_large_crops: int, large_size: int, n_small_crops: int, small_size: int, strong: bool = True)`
   - Stores `self.dataset = dataset`, `self.n_large_crops = n_large_crops`, `self.n_small_crops = n_small_crops`
   - Creates `self.large_aug = ContrastiveAugmentation(size=large_size, strong=strong)`
   - Creates `self.small_aug = ContrastiveAugmentation(size=small_size, strong=strong)`
   - `__len__` returns `len(self.dataset)`
   - `__getitem__(self, idx)`: get `img, label = self.dataset[idx]`, build `crops = []`, append `self.large_aug(img)` n_large_crops times, append `self.small_aug(img)` n_small_crops times, return `(crops, label)`
   - NOTE: The base dataset must return (PIL_Image, label) — NOT already-transformed tensors. MultiCropDataset applies its own augmentation.

2. `ssl_collate_multi_crop(batch)`:
   - Input: list of `(crops_list, label)` tuples
   - `all_crops, labels = zip(*batch)`
   - `n_crops = len(all_crops[0])`
   - `crops_list = [torch.stack([sample[i] for sample in all_crops]) for i in range(n_crops)]`
   - Return `(crops_list, torch.tensor(labels, dtype=torch.long))`
   - Returns a LIST of tensors (not a single stacked tensor) because large and small crops have different spatial dims

3. Update `SSLDataModule`:
   - Add `dataset` parameter to `__init__`: `dataset: torch.utils.data.Dataset | None = None`
   - In `setup()`: if `self.dataset is not None`, set `self.train_dataset = self.dataset` (skip ImageFolder creation and transform wrapping)
   - In `train_dataloader()`: detect if `self.train_dataset` is an instance of `MultiCropDataset` — if so, use `collate_fn=ssl_collate_multi_crop`; otherwise use `ssl_collate_fn` as before
   - Detection: `isinstance(self.train_dataset, MultiCropDataset)`

4. Update `core/data.py` module docstring to mention MultiCropDataset.

5. Write `tests/test_multi_crop.py` with all 6 behaviors above. Use a synthetic ImageFolder (tmp_path with PIL images) as the base dataset. Use `torchvision.datasets.ImageFolder` without transform as the base dataset passed to MultiCropDataset.
