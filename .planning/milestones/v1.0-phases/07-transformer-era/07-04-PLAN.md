   **Module-level docstring:**
   ```
   DINOv2 Feature Extraction and Evaluation Tutorial.

   Demonstrates using pretrained DINOv2 (Oquab et al., 2023) features for:
   1. Zero-shot k-NN classification
   2. Linear probing

   DINOv2 is loaded as a feature extractor via timm -- no from-scratch training
   (requires hundreds of GPU-days and the LVD-142M dataset).

   Note: Register tokens (added Oct 2023) change the model API. Use the checkpoint
   version that matches your timm installation. The correct lineage is:
   DINO -> DINOv2 -> DINOv2 + Registers. "DINOv3" does not exist.

   Usage:
       python eval/dinov2_demo.py --dataset cifar10
       python eval/dinov2_demo.py --dataset stl10
       python eval/dinov2_demo.py --dataset imagefolder --data-dir /path/to/data
   ```

   **Imports:** argparse, torch, torch.nn, timm, torchvision.datasets (CIFAR10, STL10, ImageFolder), torchvision.transforms, torch.utils.data.DataLoader, sklearn.neighbors.KNeighborsClassifier, sklearn.linear_model.SGDClassifier (or torch.nn.Linear for linear probing)

   **argparse (D-04):**
   - `--dataset`: choices=['cifar10', 'stl10', 'imagefolder'], default='cifar10'
   - `--data-dir`: type=str, default='data'
   - `--batch-size`: type=int, default=256
   - `--k`: type=int, default=20, help='k for k-NN'
   - `--device`: type=str, default='cuda' if torch.cuda.is_available() else 'cpu'

   **Model loading (D-03):**
   ```python
   model = timm.create_model('vit_small_patch14_dinov2.lvd142m', pretrained=True, num_classes=0)
   model = model.eval().to(device)
   ```

   **Dataset loading:**
   - cifar10: `CIFAR10(root=data_dir, train=True/False, download=True, transform=transform)` with transform = Resize(224) + CenterCrop(224) + ToTensor() + Normalize(IMAGENET_MEAN, IMAGENET_STD)
   - stl10: `STL10(root=data_dir, split='train'/'test', download=True, transform=transform)`
   - imagefolder: `ImageFolder(root=data_dir, transform=transform)`

   **Feature extraction function:**
   ```python
   def extract_features(model, dataloader, device):
       features, labels = [], []
       with torch.no_grad():
           for imgs, lbls in dataloader:
               feats = model(imgs.to(device))
               features.append(feats.cpu())
               labels.append(lbls)
       return torch.cat(features).numpy(), torch.cat(labels).numpy()
   ```

   **k-NN evaluation:**
   ```python
   knn = KNeighborsClassifier(n_neighbors=args.k, metric='cosine')
   knn.fit(train_features, train_labels)
   knn_acc = knn.score(test_features, test_labels)
   print(f"k-NN accuracy (k={args.k}): {knn_acc:.4f}")
   ```

   **Linear probing:**
   - Use a simple torch.nn.Linear trained with SGD for 100 epochs on extracted features
   - Or use sklearn SGDClassifier for simplicity
   - Print accuracy:
   ```python
   print(f"Linear probe accuracy: {probe_acc:.4f}")
   ```

   **Main guard:** `if __name__ == "__main__": main()`
