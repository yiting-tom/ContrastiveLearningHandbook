### Task 5.1: Write stage-1 YAML config

**File:** `configs/supcon_stage1_resnet18.yaml`
**Action:** Create new file.

**Description:**

Follows the exact layout of `configs/simclr_v1_resnet18.yaml`. Header comment block must include the sum-outside gotcha, class-sampler requirement, and the two-stage training note (DOC-02 requirement for configs).

```yaml
# Supervised Contrastive Learning — Stage 1 (Pretraining)
# Khosla et al., NeurIPS 2020 — "Supervised Contrastive Learning"
# arXiv: https://arxiv.org/abs/2004.11362
#
# STAGE 1 of 2: Pretrain backbone + projection head with SupCon loss.
# No classifier is used in this stage.
#
# ── Two-stage workflow ─────────────────────────────────────────────────────
#   Stage 1 (this config):
#       python train.py --config configs/supcon_stage1_resnet18.yaml
#
#   Stage 2 (linear fine-tuning):
#       python train.py --config configs/supcon_stage2_resnet18.yaml \
#           --ckpt_path logs/supcon/version_0/checkpoints/last.ckpt
#
# ── Key implementation notes ───────────────────────────────────────────────
# Sum-outside formulation (Eq. 2, NOT Eq. 1):
#   Loss = -1/|P(i)| * Σ_{p∈P(i)} [s_{ip}/τ − log Σ_{a≠i} exp(s_{ia}/τ)]
#   The division by |P(i)| is OUTSIDE the log. Eq. 1 (inside the log) is
#   mathematically valid but empirically weaker. SupConLoss implements Eq. 2.
#
# Class-balanced sampler REQUIRED:
#   Without ClassBalancedSampler, many classes are singletons in a batch and
#   have zero supervised positives. The SupCon loss degrades to SimCLR.
#   n_classes_per_batch * n_samples_per_class should equal batch_size.
#   Default: 128 classes × 2 samples = 256 batch size.
#
# Do NOT add a classifier in stage 1:
#   Mixing CE loss with SupCon during pretraining collapses the contrastive
#   representation (warned against in the SupCon paper). Use this config for
#   stage 1 only, then switch to supcon_stage2_resnet18.yaml.

method: supcon
backbone: resnet18
pretrained: false

max_epochs: 200
warmup_epochs: 10
batch_size: 256
lr: 0.5
weight_decay: 1.0e-4
optimizer: lars
n_views: 2
data_dir: data
num_workers: 4

supcon:
  temperature: 0.07
  n_samples_per_class: 2
  n_classes_per_batch: 128   # 128 classes × 2 samples/class = 256 batch size
  num_classes: 10            # used only by stage-2 module (SupConFinetuneModule)
  projection_dim: 128
```

### Task 5.2: Write stage-2 YAML config

**File:** `configs/supcon_stage2_resnet18.yaml`
**Action:** Create new file.

**Description:**

Stage-2 config references `method: supcon_finetune`. Uses SGD, `weight_decay: 0.0`, `n_views: 1` (single view for CE training), `warmup_epochs: 0`.

```yaml
# Supervised Contrastive Learning — Stage 2 (Linear Fine-tuning)
# Khosla et al., NeurIPS 2020 — "Supervised Contrastive Learning"
# arXiv: https://arxiv.org/abs/2004.11362
#
# STAGE 2 of 2: Freeze backbone from stage 1; train linear head with SGD.
# Load stage-1 checkpoint via --ckpt_path argument.
#
# ── Two-stage workflow ─────────────────────────────────────────────────────
#   Stage 1 (backbone pretraining):
#       python train.py --config configs/supcon_stage1_resnet18.yaml
#
#   Stage 2 (this config):
#       python train.py --config configs/supcon_stage2_resnet18.yaml \
#           --ckpt_path logs/supcon/version_0/checkpoints/last.ckpt
#
# ── Key implementation notes ───────────────────────────────────────────────
# weight_decay MUST be 0.0:
#   The linear probe is a single layer applied to frozen features. Nonzero
#   weight decay regularizes away the linear classifier's signal. SGD with
#   momentum=0.9 and weight_decay=0.0 is the standard linear probe setup
#   from the SupCon paper (Table 5).
#
# ClassBalancedSampler NOT used in stage 2:
#   Stage 2 trains a CE classifier, not a contrastive loss. Standard random
#   shuffle is correct. The class-balanced sampler is only needed in stage 1.
#
# n_views: 1 for stage 2:
#   Multi-view augmentation is unnecessary for cross-entropy training on a
#   frozen backbone. Using 1 view halves the data loading cost.

method: supcon_finetune
backbone: resnet18
pretrained: false

max_epochs: 100
warmup_epochs: 0
batch_size: 256
lr: 0.1
weight_decay: 0.0
optimizer: sgd
n_views: 1
data_dir: data
num_workers: 4

supcon:
  temperature: 0.07
  n_samples_per_class: 2
  n_classes_per_batch: 128
  num_classes: 10            # REQUIRED: must match the number of classes in the dataset
  projection_dim: 128
```

### Task 5.3: Add `get_method` helper to dispatcher

**File:** `core/dispatcher.py`
**Action:** Edit — append after `available_methods()`.

**Description:**

Plans 03 and 05 verification commands use `get_method("supcon")` to look up a class by name. Add this helper:

```python
def get_method(name: str) -> type[BaseSSLModule]:
    """Look up a registered method class by name.

    Args:
        name: Method name as registered with register_method().

    Returns:
        The BaseSSLModule subclass registered under name.

    Raises:
        ValueError: If name is not registered.
    """
    if name not in _METHOD_REGISTRY:
        available = sorted(_METHOD_REGISTRY.keys())
        raise ValueError(
            f"Unknown method: '{name}'. Available methods: {available}"
        )
    return _METHOD_REGISTRY[name]
```

```python
"""
Smoke test: SupCon stage-1 trains for 3 epochs on synthetic data.

Run with:
    python -m pytest tests/test_supcon_smoke.py -v -s
or directly:
    python tests/test_supcon_smoke.py
"""
# File: tests/test_supcon_smoke.py

import os
import tempfile

import lightning as L
import torch
from torch.utils.data import DataLoader

from core.config import TrainConfig
from core.data import ClassBalancedSampler, SSLDataModule, ssl_collate_fn
from core.dispatcher import get_method


class FakeTinyDataset(torch.utils.data.Dataset):
    """Synthetic 4-class dataset for smoke testing. No disk I/O required."""

    def __init__(self, n_per_class: int = 20, n_classes: int = 4, n_views: int = 2):
        self.targets = [c for c in range(n_classes) for _ in range(n_per_class)]
        self.n_views = n_views

    def __len__(self):
        return len(self.targets)

    def __getitem__(self, idx):
        label = self.targets[idx]
        # Random 3×32×32 image — values don't matter for smoke test
        views = [torch.randn(3, 32, 32) for _ in range(self.n_views)]
        return views, label


def test_supcon_stage1_three_epochs():
    """Stage-1 SupCon runs 3 epochs without NaN loss on synthetic data."""
    import methods.supcon  # trigger dispatcher registration

    cfg = TrainConfig.model_validate({
        "method": "supcon",
        "backbone": "resnet18",
        "pretrained": False,
        "max_epochs": 3,
        "warmup_epochs": 0,
        "batch_size": 8,
        "lr": 1e-3,
        "weight_decay": 1e-4,
        "optimizer": "adamw",
        "n_views": 2,
        "data_dir": "data",
        "num_workers": 0,
        "supcon": {
            "temperature": 0.07,
            "n_samples_per_class": 2,
            "n_classes_per_batch": 4,
            "num_classes": 4,
            "projection_dim": 128,
        },
    })

    SupConModule = get_method("supcon")
    module = SupConModule(cfg)

    # Build synthetic dataset with ClassBalancedSampler
    dataset = FakeTinyDataset(n_per_class=10, n_classes=4, n_views=2)
    sampler = ClassBalancedSampler(
        dataset,
        n_classes_per_batch=4,
        n_samples_per_class=2,
    )
    loader = DataLoader(
        dataset,
        batch_size=8,
        sampler=sampler,
        shuffle=False,
        collate_fn=ssl_collate_fn,
        drop_last=True,
    )

    with tempfile.TemporaryDirectory() as tmpdir:
        trainer = L.Trainer(
            max_epochs=3,
            accelerator="cpu",
            enable_checkpointing=False,
            logger=False,
            enable_progress_bar=True,
            default_root_dir=tmpdir,
        )
        trainer.fit(module, train_dataloaders=loader)

    # Verify the loss logged during training was finite
    train_losses = [
        v for k, v in trainer.logged_metrics.items()
        if "train/loss" in k or "train_loss" in k
    ]
    if train_losses:
        assert all(torch.isfinite(torch.tensor(v)) for v in train_losses), (
            f"NaN/Inf loss detected: {train_losses}"
        )
    print("Stage-1 3-epoch smoke test PASSED.")


if __name__ == "__main__":
    test_supcon_stage1_three_epochs()
```
