"""t-SNE visualization of SSL representations.

Runs PCA pre-reduction to 50 dimensions, then t-SNE with cosine metric.
Sweeps multiple perplexity values and saves one PNG per perplexity.

WARNING: t-SNE preserves local structure only. Do NOT interpret global
distances or cluster spacing. Different perplexity values emphasize
different scales of structure -- always compare multiple values.

Usage:
    python eval/tsne_vis.py configs/simclr_v1_resnet18.yaml --ckpt outputs/run/checkpoints/epoch-99.ckpt
"""
from __future__ import annotations
import argparse
from pathlib import Path
import yaml
import torch
import torch.nn.functional as F
import numpy as np
import matplotlib
matplotlib.use("Agg")
import matplotlib.pyplot as plt
from sklearn.decomposition import PCA
from sklearn.manifold import TSNE

from core.config import TrainConfig, TSNEConfig
from core.dispatcher import get_method
```

Implementation:

1. `extract_features(backbone, dataloader, n_samples, device) -> tuple[np.ndarray, np.ndarray]`:
   - Iterate dataloader collecting features via `backbone(imgs.to(device))` and labels (handle multi-view: `imgs = batch[0][0] if batch[0].ndim == 5 else batch[0]`, `labels = batch[1]`).
   - Stop after collecting >= n_samples. Cat, L2-normalize, subsample to exactly n_samples if over.
   - Return `(features.cpu().numpy(), labels.cpu().numpy())`.

2. `run_tsne(features: np.ndarray, labels: np.ndarray, perplexities: list[int], output_dir: Path) -> list[Path]`:
   - `output_dir.mkdir(parents=True, exist_ok=True)`.
   - PCA pre-reduction: if features.shape[1] > 50: `features = PCA(n_components=50, random_state=42).fit_transform(features)`.
   - For each perplexity in perplexities:
     - `embedding = TSNE(n_components=2, perplexity=perplexity, init='pca', metric='cosine', learning_rate='auto', random_state=42).fit_transform(features)`.
     - Create scatter plot: `fig, ax = plt.subplots(figsize=(8, 8))`, `scatter = ax.scatter(embedding[:, 0], embedding[:, 1], c=labels, cmap='tab10', s=5, alpha=0.7)`, `ax.set_title(f"t-SNE (perplexity={perplexity})")`, `plt.colorbar(scatter, ax=ax)`, `ax.set_xticks([])`, `ax.set_yticks([])`.
     - Save: `path = output_dir / f"tsne_perp{perplexity}.png"`, `fig.savefig(path, dpi=150, bbox_inches='tight')`, `plt.close(fig)`.
   - Return list of saved paths.

3. `get_args()`: positional `config`, `--ckpt` required, `--output-dir` optional (default: `{ckpt_parent_parent}/tsne/`), `--device`.

4. `main()`: load config, import methods, load checkpoint, extract features, run t-SNE, print saved paths.

5. `if __name__ == "__main__": main()`

"""UMAP visualization of SSL representations.

Preferred over t-SNE as the primary visualization: faster, preserves more
global structure, and the reducer object can map new samples.

Usage:
    python eval/umap_vis.py configs/simclr_v1_resnet18.yaml --ckpt outputs/run/checkpoints/epoch-99.ckpt
"""
from __future__ import annotations
import argparse
from pathlib import Path
import yaml
import torch
import torch.nn.functional as F
import numpy as np
import matplotlib
matplotlib.use("Agg")
import matplotlib.pyplot as plt
import umap

from core.config import TrainConfig, UMAPConfig
from core.dispatcher import get_method
```

Implementation:

1. `extract_features(backbone, dataloader, n_samples, device) -> tuple[np.ndarray, np.ndarray]`:
   - Same pattern as tsne_vis.py: iterate, collect, normalize, subsample to n_samples.

2. `run_umap(features: np.ndarray, labels: np.ndarray, metric: str, output_dir: Path) -> tuple[Path, umap.UMAP]`:
   - `output_dir.mkdir(parents=True, exist_ok=True)`.
   - Print torchdr note if `features.shape[0] > 50_000`: `print("Note: For datasets >50K samples, consider torchdr for GPU-accelerated UMAP: pip install torchdr")`.
   - `reducer = umap.UMAP(metric=metric, random_state=42, n_components=2)`.
   - `embedding = reducer.fit_transform(features)`.
   - Plot: `fig, ax = plt.subplots(figsize=(8, 8))`, `scatter = ax.scatter(embedding[:, 0], embedding[:, 1], c=labels, cmap='tab10', s=5, alpha=0.7)`, `ax.set_title("UMAP")`, `plt.colorbar(scatter, ax=ax)`, `ax.set_xticks([])`, `ax.set_yticks([])`.
   - Save: `path = output_dir / "umap.png"`, `fig.savefig(path, dpi=150, bbox_inches='tight')`, `plt.close(fig)`.
   - Return `(path, reducer)`.

3. `get_args()`: positional `config`, `--ckpt` required, `--output-dir` optional (default: `{ckpt_parent_parent}/umap/`), `--device`.

4. `main()`: load config, import methods, load checkpoint, get umap_cfg from `cfg.eval.umap if cfg.eval and cfg.eval.umap else UMAPConfig()`, extract features, run UMAP, print path.

5. `if __name__ == "__main__": main()`
