     - Always L2-normalize inputs: `z_i = F.normalize(z_i, dim=1)`, `z_j = F.normalize(z_j, dim=1)`
     - **Symmetric mode (queue=None):** Concatenate `z = cat([z_i, z_j], dim=0)` -> `[2B, D]`. Compute similarity matrix `sim = z @ z.T / temperature` -> `[2B, 2B]`. Mask diagonal with `-inf`. Labels: `cat([arange(B, 2B), arange(0, B)])`. Return `F.cross_entropy(sim, labels, reduction=self.reduction)`.
     - **Asymmetric mode (queue provided):** queue shape is `[D, K]`. `l_pos = (z_i * z_j).sum(dim=1, keepdim=True) / temperature` -> `[B, 1]`. `l_neg = z_i @ queue / temperature` -> `[B, K]`. `logits = cat([l_pos, l_neg], dim=1)` -> `[B, 1+K]`. Labels: `zeros(B, dtype=long)`. Return `F.cross_entropy(logits, labels, reduction=self.reduction)`.
   - Use `F.cross_entropy` (handles log-sum-exp internally for numerical stability, per Pitfall 7).

   - Class docstring: "Layer-wise Adaptive Rate Scaling optimizer. Reference: https://arxiv.org/abs/1708.03888"
   - `__init__(self, params, lr, momentum=0.9, weight_decay=1e-6, eta=0.001, exclude_bias_and_norm=True)` (per D-05)

   - `@torch.no_grad() def step(self, closure=None)`:
     - For each param group, for each param with non-None grad:
       - Apply weight decay: `grad = grad.add(p, alpha=weight_decay)` (if weight_decay != 0)

       - Update: `p.add_(grad, alpha=-scaled_lr)`
