    **tools/visualize_augmentations.py** — Standalone CLI script (per D-03):

    ```python
    """Visualize the SSL augmentation pipeline.

    Standalone CLI script that applies ContrastiveAugmentation to a single
    image and saves a grid of augmented views. Use this to verify that
    strong color jitter (s=1.0) and Gaussian blur are present in the
    SimCLR augmentation pipeline.

    Usage:
        python tools/visualize_augmentations.py --data-dir data --n-views 8
        python tools/visualize_augmentations.py --output tools/output/augmentation_grid.png
    """
    ```

    Implementation details:
    1. `import argparse, pathlib, matplotlib, torch`
    2. `matplotlib.use("Agg")` (non-interactive backend — no display needed)
    3. `import matplotlib.pyplot as plt`
    4. `from torchvision.datasets import ImageFolder`
    5. `from core.data import ContrastiveAugmentation`
    6. Parse args: `--data-dir` (default "data"), `--output` (default "tools/output/augmentation_grid.png"), `--n-views` (default 8), `--size` (default 224), `--strong/--no-strong` (default True)
    7. Load first image from ImageFolder dataset: `dataset = ImageFolder(args.data_dir)`, `img, _ = dataset[0]` (PIL Image)
    8. Create augmentation: `aug = ContrastiveAugmentation(size=args.size, n_views=1, strong=args.strong)`
    9. Generate n_views augmented versions by calling aug(img) n_views times
    10. Create a 2-row x 4-col grid (for n_views=8) using matplotlib subplots
    11. For each augmented view: denormalize from ImageNet mean/std, clamp to [0, 1], convert to numpy, show with `ax.imshow()`
    12. Set title on first subplot: f"Strong={args.strong}, s={'1.0' if args.strong else '0.4'}"
    13. `plt.tight_layout()`, create output directory with `pathlib.Path(args.output).parent.mkdir(parents=True, exist_ok=True)`, save figure
    14. `print(f"Saved augmentation grid to {args.output}")`

    For denormalization, use `IMAGENET_MEAN = [0.485, 0.456, 0.406]` and `IMAGENET_STD = [0.229, 0.224, 0.225]` (standard values; import from core.data if exported, otherwise define locally).

    The script must work as `python tools/visualize_augmentations.py --data-dir <path>` without any package installation.
