```python
"""Linear probe evaluation for SSL checkpoints.

Freezes the pretrained backbone, pre-extracts features (cached to disk),
and trains a linear classifier head with SGD (weight_decay=0.0).

Usage:
    python eval/linear_probe.py configs/simclr_v1_resnet18.yaml --ckpt outputs/run/checkpoints/epoch-99.ckpt
"""
from __future__ import annotations
import argparse
from pathlib import Path

import torch
import torch.nn as nn
import torch.nn.functional as F
import lightning as L
from torch.optim import SGD
from torch.optim.lr_scheduler import MultiStepLR
from torch.utils.data import DataLoader, TensorDataset
import yaml

from core.config import TrainConfig, LinearProbeConfig
from core.dispatcher import get_method
```

Key implementation details:

1. `extract_and_cache(backbone, dataloader, cache_dir: Path, split: str, device: str, ckpt_path: str) -> tuple[Tensor, Tensor]`:
   - **Derive checkpoint stem for cache keying (D-04):** `ckpt_stem = Path(ckpt_path).stem`
   - `feat_path = cache_dir / f"{ckpt_stem}_features_{split}.pt"`, `label_path = cache_dir / f"{ckpt_stem}_labels_{split}.pt"`
   - If both exist, `return torch.load(feat_path, weights_only=True), torch.load(label_path, weights_only=True)`
   - Otherwise: `cache_dir.mkdir(parents=True, exist_ok=True)`, iterate dataloader with `@torch.no_grad()`, call `backbone(imgs.to(device))`, collect features/labels, cat, L2-normalize: `features = F.normalize(torch.cat(all_features), dim=1)`, `labels = torch.cat(all_labels)`, save both to the checkpoint-keyed paths, return both.
   - Handle SSL multi-view batches: `imgs = batch[0][0] if batch[0].ndim == 5 else batch[0]`, `labels = batch[1]`.

2. `class LinearProbeModule(L.LightningModule)`:
   - `__init__(self, feat_dim: int, num_classes: int, lp_cfg: LinearProbeConfig)`: `self.linear = nn.Linear(feat_dim, num_classes)`, `self.criterion = nn.CrossEntropyLoss()`, `self.lp_cfg = lp_cfg`, `self.save_hyperparameters()`.
   - `training_step(self, batch, batch_idx)`: `features, labels = batch`, `logits = self.linear(features)`, `loss = self.criterion(logits, labels)`, `acc = (logits.argmax(1) == labels).float().mean()`, `self.log("train/loss", loss)`, `self.log("train/acc", acc, prog_bar=True)`, return loss.
   - `validation_step(self, batch, batch_idx)`: same but log `val/loss` and `val/acc`.
   - `configure_optimizers(self)`: `optimizer = SGD(self.linear.parameters(), lr=self.lp_cfg.lr, weight_decay=0.0)`, `scheduler = MultiStepLR(optimizer, milestones=self.lp_cfg.milestones)`, `return [optimizer], [scheduler]`. Add `assert optimizer.param_groups[0]["weight_decay"] == 0.0, "Linear probe requires weight_decay=0.0"`.

3. `def get_args() -> argparse.Namespace`: positional `config` arg, `--ckpt` required, `--output-dir` optional (default None -> cache derived from ckpt path), `--device` (default auto).

4. `def main()`:
   - Parse args, load config via `TrainConfig.model_validate(yaml.safe_load(open(args.config)))`.
   - `import methods` (trigger dispatcher registration).
   - `MethodClass = get_method(cfg.method)`, `model = MethodClass.load_from_checkpoint(args.ckpt, cfg=cfg)`, `model.eval()`, `model.to(args.device)`, `model.backbone.requires_grad_(False)`.
   - Resolve cache_dir: `cache_dir = Path(args.ckpt).parent.parent / "cache"` (per D-04).
   - Build dataloaders from SSLDataModule: `from core.data import SSLDataModule`, `dm = SSLDataModule(cfg)`, `dm.setup("fit")`.
   - Extract features with checkpoint-keyed caching: `train_feats, train_labels = extract_and_cache(model.backbone, dm.train_dataloader(), cache_dir, "train", args.device, args.ckpt)`, same for val.
   - Infer `num_classes = int(train_labels.max().item()) + 1`, `feat_dim = train_feats.shape[1]`.
   - Create TensorDataset + DataLoader for cached features.
   - `lp_cfg = cfg.eval.linear_probe if cfg.eval and cfg.eval.linear_probe else LinearProbeConfig()`.
   - Create `LinearProbeModule(feat_dim, num_classes, lp_cfg)`.
   - Train with `L.Trainer(max_epochs=lp_cfg.max_epochs, accelerator="auto", logger=True)`.
   - Print final `val/acc`.

5. `if __name__ == "__main__": main()`
