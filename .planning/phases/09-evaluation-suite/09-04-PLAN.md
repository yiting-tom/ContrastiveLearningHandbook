```python
"""Fine-tuning evaluation for SSL checkpoints.

Unfreezes the backbone and trains the full model with differentiated
learning rates: backbone at a lower LR to prevent catastrophic forgetting,
classification head at a higher LR.

Usage:
    python eval/finetune.py configs/simclr_v1_resnet18.yaml --ckpt outputs/run/checkpoints/epoch-99.ckpt
"""
from __future__ import annotations
import argparse
import math
from pathlib import Path
import yaml
import torch
import torch.nn as nn
import lightning as L
from torch.optim import AdamW
from torch.optim.lr_scheduler import LambdaLR

from core.config import TrainConfig, FinetuneConfig
from core.data import SSLDataModule
from core.dispatcher import get_method
```

Implementation:

1. `class FinetuneModule(L.LightningModule)`:
   - `__init__(self, backbone: nn.Module, feat_dim: int, num_classes: int, ft_cfg: FinetuneConfig, max_epochs: int = 100, warmup_epochs: int = 10)`:
     - Store backbone, create `self.head = nn.Linear(feat_dim, num_classes)`, `self.criterion = nn.CrossEntropyLoss()`, store ft_cfg, max_epochs, warmup_epochs.
     - `self.save_hyperparameters(ignore=["backbone"])`.
   - `forward(self, x)`: `features = self.backbone(x)`, `return self.head(features)`.
   - `training_step(self, batch, batch_idx)`: `imgs, labels = batch[0], batch[1]` (handle multi-view: `imgs = imgs[0] if imgs.ndim == 5 else imgs`), `logits = self(imgs)`, loss, acc, log `train/loss`, `train/acc`.
   - `validation_step(self, batch, batch_idx)`: same, log `val/loss`, `val/acc`.
   - `configure_optimizers(self)`:
     - `param_groups = [{"params": self.backbone.parameters(), "lr": self.ft_cfg.backbone_lr}, {"params": self.head.parameters(), "lr": self.ft_cfg.head_lr}]`
     - `optimizer = AdamW(param_groups, weight_decay=1e-4)`
     - Warmup-cosine lambda scheduler:
       ```python
       total_steps = self.trainer.estimated_stepping_batches
       warmup_steps = int(total_steps * self.warmup_epochs / self.max_epochs)
       def lr_lambda(step):
           if step < warmup_steps:
               return step / max(1, warmup_steps)
           progress = (step - warmup_steps) / max(1, total_steps - warmup_steps)
           return 0.5 * (1 + math.cos(math.pi * progress))
       scheduler = LambdaLR(optimizer, lr_lambda)
       ```
     - Return `[optimizer], [{"scheduler": scheduler, "interval": "step"}]`
   - `def train(self, mode=True)`: Override to keep BN in eval when freeze_bn=True:
     ```python
     super().train(mode)
     if self.ft_cfg.freeze_bn and mode:
         for m in self.backbone.modules():
             if isinstance(m, (nn.BatchNorm1d, nn.BatchNorm2d, nn.SyncBatchNorm)):
                 m.eval()
     return self
     ```

2. `get_args()`: positional `config`, `--ckpt` required, `--output-dir` optional, `--device`.

3. `main()`:
   - Load config, import methods, load checkpoint via `get_method(cfg.method).load_from_checkpoint(args.ckpt, cfg=cfg)`.
   - `backbone = model.backbone`, `feat_dim = backbone.num_features` (from timm).
   - Build data module: `dm = SSLDataModule(cfg)`, `dm.setup("fit")`.
   - Infer num_classes from val dataset class count.

   - Create FinetuneModule, train with `L.Trainer(max_epochs=cfg.max_epochs, ...)`.
   - Print final val/acc.

4. `if __name__ == "__main__": main()`

```python
"""Class Activation Map (CAM) visualization for SSL checkpoints.

Uses EigenCAM by default (gradient-free, works without a classifier --
correct default for SSL models). Switches to GradCAM when a downstream
classifier is present.

Architecture-aware target layer selection:
  - ResNet: backbone.layer4[-1]
  - ViT: backbone.blocks[-1].norm1 (with reshape_transform)

Usage:
    python eval/cam_vis.py configs/simclr_v1_resnet18.yaml --ckpt outputs/run/checkpoints/epoch-99.ckpt
    python eval/cam_vis.py configs/simclr_v1_resnet18.yaml --ckpt outputs/run/checkpoints/epoch-99.ckpt --classifier outputs/finetune/model.ckpt
"""
from __future__ import annotations
import argparse
from pathlib import Path
import yaml
import numpy as np
import torch
import torch.nn.functional as F
import matplotlib
matplotlib.use("Agg")
import matplotlib.pyplot as plt
from PIL import Image
from torchvision import transforms

from pytorch_grad_cam import EigenCAM, GradCAM
from pytorch_grad_cam.utils.image import show_cam_on_image

from core.config import TrainConfig, CAMConfig
from core.dispatcher import get_method
```

Implementation:

1. `get_target_layer(backbone: nn.Module, backbone_name: str) -> list[nn.Module]`:
   - If `"resnet" in backbone_name.lower()`: return `[backbone.layer4[-1]]`.
   - If `"vit" in backbone_name.lower()`: return `[backbone.blocks[-1].norm1]`.
   - Else: `raise ValueError(f"Unsupported backbone for CAM: {backbone_name}. Supported: resnet*, vit*")`.

2. `vit_reshape_transform(tensor: torch.Tensor, height: int = 14, width: int = 14) -> torch.Tensor`:
   - Remove CLS token: `result = tensor[:, 1:, :]`.
   - Reshape: `result = result.reshape(tensor.size(0), height, width, tensor.size(2))`.
   - Permute to BCHW: `result = result.permute(0, 3, 1, 2)`.
   - Return result.

3. `get_cam_method(method_name: str)`:
   - Map: `{"eigencam": EigenCAM, "gradcam": GradCAM}`.
   - Return the class. Raise ValueError if unknown.

4. `run_cam(model, backbone_name: str, images: list[np.ndarray], cam_cfg: CAMConfig, output_dir: Path, classifier: nn.Module | None = None) -> list[Path]`:
   - `output_dir.mkdir(parents=True, exist_ok=True)`.
   - `target_layers = get_target_layer(model.backbone, backbone_name)`.
   - `reshape_transform = vit_reshape_transform if "vit" in backbone_name.lower() else None`.
   - Determine CAM class: `CamClass = get_cam_method(cam_cfg.method)`.
   - If classifier and cam_cfg.method == "gradcam": wrap model + classifier for GradCAM. If no classifier and method is "gradcam": warn and fall back to EigenCAM.
   - For EigenCAM: `cam = CamClass(model=model.backbone, target_layers=target_layers, reshape_transform=reshape_transform)`.
   - Preprocess each image: resize to 224x224, normalize with ImageNet stats, convert to tensor.
   - For each image: `grayscale_cam = cam(input_tensor=input_tensor, targets=None)`, `cam_image = show_cam_on_image(rgb_img_float, grayscale_cam[0], use_rgb=True)`, save with `plt.imsave(output_dir / f"cam_{i:02d}.png", cam_image)`.
   - Return list of saved paths.

5. `get_args()`: positional `config`, `--ckpt` required, `--classifier` optional, `--output-dir`, `--device`.

6. `main()`:
   - Load config, import methods, load checkpoint.
   - Get cam_cfg from config or default.
   - Load reference images: either from val dataset (first n_images images) or from data_dir.
   - For simplicity, use val dataset images: `dm = SSLDataModule(cfg)`, `dm.setup("fit")`, take first n_images from val_dataset (or train_dataset if no val).
   - Convert dataset images to numpy RGB arrays (denormalize or load raw).
   - Alternative simpler approach: load raw images from data_dir using PIL, not through the dataloader.
   - Run CAM, print saved paths.

7. `if __name__ == "__main__": main()`
